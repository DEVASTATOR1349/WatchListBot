# 🎬 WatchList Bot

Telegram-бот для ведения очереди фильмов, сериалов, аниме и мультфильмов.
Поддерживает личные и **общие групповые списки** (для семьи, друзей).

> Бот: [@FilmANDscbot](https://t.me/FilmANDscbot)

---

## 🚀 Возможности

- **➕ Добавление** — название, категория (🎬 фильм / 📺 сериал / 🌸 аниме / 🧸 мультфильм / 📦 прочее), мультивыбор жанров, заметка
- **📋 Список** — пагинация, фильтр по статусу (в планах / смотрю / просмотрено)
- **✅ Статусы** — запланировано → смотрю → просмотрено → оценка 1–10
- **👥 Группы** — общие списки по инвайт-кодам (8 символов), можно делить с женой/друзьями
- **📊 Статистика** — по категориям, по периодам (неделя/месяц/всё), отдельно для личного и групп
- **🗑 Удаление** — автор записи или любой участник группы может удалить

## 🧱 Стек

- Python 3.12
- [aiogram 3.x](https://docs.aiogram.dev/) — Telegram Bot API
- [Peewee](http://docs.peewee-orm.com/) — ORM + SQLite
- Docker

## ⚙️ Установка и запуск

### 1. Клонировать

```bash
git clone https://github.com/DEVASTATOR1349/WatchListBot.git
cd WatchListBot
```

### 2. Настроить `.env`

```bash
cp .env.example .env
```

Заполни:

| Переменная           | Описание                                      |
|----------------------|-----------------------------------------------|
| `TELEGRAM_BOT_TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) |
| `TG_PROXY`           | Прокси для Telegram API (опционально)         |
| `ADMIN_TELEGRAM_ID`  | Твой Telegram ID (опционально)                |

### 3. Запустить через Docker

```bash
docker build -t watchlist-bot .
docker run -d --name watchlist-bot --restart unless-stopped watchlist-bot
```

### 4. Без Docker

```bash
pip install -r requirements.txt
python bot.py
```

## 📖 Как пользоваться

### Основное

| Команда / Кнопка    | Что делает                            |
|----------------------|---------------------------------------|
| `/start`             | Главное меню со статистикой           |
| `➕ Добавить`        | Добавить фильм/сериал в очередь       |
| `🎬 Мой список`      | Просмотр списка с фильтрами           |
| `📊 Статистика`      | Статистика по категориям и датам      |
| `👥 Группы`          | Управление групповыми списками        |
| `🏠 Главная`         | Вернуться на главную                  |

### Группы

1. Нажми **👥 Группы** → **➕ Создать группу** → введи название
2. Получи **8-значный код** — отправь его тому, с кем хочешь делить список
3. Второй человек жмёт **👥 Группы** → **🔗 Войти по коду** → вводит код
4. Готово! Теперь при добавлении можно выбрать «личное» или «в группу»
